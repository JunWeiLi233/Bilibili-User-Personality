# B站评论标注指南 — Ziegenbein 四轴标注法

> 用于人工标注 Bilibili 评论的争辩性行为风险。请独立完成，不要参考他人标注结果。

## 四个评价轴（每个轴 0-2 分）

### 1. toxicEmotions（情绪过激 / 毒性情绪）
评论是否包含愤怒、讽刺、辱骂、人身攻击等情绪化表达？

| 分数 | 标准 | 示例 |
|------|------|------|
| **0** | 无情绪过激，理性表达 | "我觉得这个方案可以再优化一下" |
| **1** | 有明显情绪，讽刺或轻度攻击 | "笑死，你这理解能力也是没谁了" |
| **2** | 强烈攻击，辱骂或恶意嘲讽 | "sb玩意儿，脑子进水了吧？滚" |

**注意**：只评价评论文本本身的情绪，不要推测发评论的人是什么状态。

### 2. missingCommitment（回避讨论 / 缺少承诺）
评论是否回避实质性讨论？包括转移话题、拒绝提供证据、重复无意义内容？

| 分数 | 标准 | 示例 |
|------|------|------|
| **0** | 积极参与讨论，提供理由或证据 | "根据官方公告，这个功能会在下个版本上线" |
| **1** | 回避深入讨论，敷衍搪塞 | "懂的都懂，不解释了" |
| **2** | 明确拒绝讨论，恶意转移话题 | "懒得跟你们说，爱信不信" |

**注意**：正常人表示"不知道"、"没研究过"不算回避讨论。

### 3. missingIntelligibility（逻辑混乱 / 缺少可理解性）
评论是否逻辑混乱、自相矛盾、使用绝对化断言且无推理？

| 分数 | 标准 | 示例 |
|------|------|------|
| **0** | 逻辑清晰，表达通顺 | "考虑到成本和时间，我觉得先做A方案比较合理" |
| **1** | 有绝对化断言，但上下文可理解 | "这绝对是今年最烂的游戏，没有之一" |
| **2** | 严重前后矛盾，或纯情绪发泄无实质内容 | "你们都是错的因为我是对的反正就是错的" |

**注意**：网络用语、玩梗、简写不算逻辑混乱，看内容是否有可理解的推理。

### 4. otherReasons（其他问题）
评论是否有其他影响讨论质量的问题？如刷屏、无关广告、恶意引战等？

| 分数 | 标准 | 示例 |
|------|------|------|
| **0** | 无其他问题 | 正常评论 |
| **1** | 轻微引战或偏离主题 | "就这？隔壁某某UP主做得比你好一百倍" |
| **2** | 明显引战、刷屏、广告、恶意带节奏 | "加群xxxxx 免费领取xxxxx 点击链接xxxxx" |

## 标注步骤

1. **通读评论**全文，不要只看关键词
2. 依次判断 4 个轴，每个轴独立打分（0/1/2）
3. 如果犹豫是 0 还是 1，默认选 0（保守原则）
4. 标注的是**这条评论文本本身**，不是评论作者

## 三个完整标注示例

### 示例 1：
**评论**："UP主分析得很到位，但是关于第三点我有不同看法，因为官方文档里明确写了这个接口已经废弃了"

| 轴 | 分数 | 理由 |
|----|------|------|
| toxicEmotions | 0 | 理性讨论，无攻击性 |
| missingCommitment | 0 | 提出了具体论据（官方文档） |
| missingIntelligibility | 0 | 逻辑清晰 |
| otherReasons | 0 | 正常讨论 |

### 示例 2：
**评论**："哈哈哈你急了？破防了吧[藏狐] 说两句就开始人身攻击，就这水平还来辩论"

| 轴 | 分数 | 理由 |
|----|------|------|
| toxicEmotions | 1 | 有嘲笑语气（哈哈哈、就这），但未到辱骂程度 |
| missingCommitment | 1 | 主要是在嘲讽而非推进讨论 |
| missingIntelligibility | 0 | 表达通顺 |
| otherReasons | 0 | 无其他问题 |

### 示例 3：
**评论**："评论区某些人能不能别刷屏了，全是复读机，一点营养都没有，这破站迟早完蛋"

| 轴 | 分数 | 理由 |
|----|------|------|
| toxicEmotions | 1 | "破站迟早完蛋"有明显不满情绪 |
| missingCommitment | 1 | 抱怨刷屏但未提出建设性意见 |
| missingIntelligibility | 0 | 逻辑通顺 |
| otherReasons | 0 | 虽有批评但无恶意引战 |

## B站特有注意事项

- **表情包**（如 [doge]、[藏狐]、[笑哭]）可能表示反讽或缓和语气，结合上下文判断
- **梗和简写**（如 dddd、绷不住了）是社区常态，不一定代表回避讨论
- **回复 @某人**：注意区分在怼原UP主还是怼其他评论者
- **游戏/动漫语境**：一些在其他场景算攻击的词（如"逆天"、"离谱"）在游戏中可能是中性或褒义
